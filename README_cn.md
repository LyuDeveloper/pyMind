# pyMind
#### 版本: CONCEPT 1.0.0 ( Public 1 )

##### 这是什么?
一个Python算法，对小米HyperMind习惯发现功能实现的尝试

##### 它的原理?
收集用户对智能设备的操作，检查两个操作之间是否小于30秒且这两个操作是否多次同时发生来确定用户习惯

收集的信息会保存至本地

##### 为什么是 概念 版本?

功能的基本概念已经完成，还有很多东西需要完善

#### 目录结构
├─Data: 储存用户操作记录 *.json\
├─Habit: 储存已经发现的习惯 habits.json\
├─Log: 日志目录\
│  ├─Collector\
│  └─Mind
